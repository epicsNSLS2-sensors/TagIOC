**An EPICS IOC for the Texas Instruments CC1350 SensorTag using Bluetooth low energy**

Supported sensors:
- Humidity-temperature
- Pressure-temperature
- Light
- Accelerometer
- Gyroscope
- Magnetometer
- Buttons
- Preliminary support:
	- Battery
